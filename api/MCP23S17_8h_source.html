00001 /*
00002  MCP23S17.h  Version 0.1
00003  Microchip MCP23S17 SPI I/O Expander Class for Arduino
00004  Created by Cort Buffington & Keith Neufeld
00005  March, 2011
00006 
00007  Features Implemented (by word and bit):
00008  I/O Direction
00009  Pull-up on/off
00010  Input inversion
00011  Output write
00012  Input read
00013 
00014  Interrupt features are not implemented in this version
00015  byte based (portA, portB) functions are not implemented in this version
00016 
00017  NOTE:  Addresses below are only valid when IOCON.BANK=0 (register addressing mode)
00018  This means one of the control register values can change register addresses!
00019  The default values is 0, so that's how we're using it.
00020 
00021  All registers except ICON (0xA and 0xB) are paired as A/B for each 8-bit GPIO port.
00022  Comments identify the port's name, and notes on how it is used.
00023 
00024  *THIS CLASS ENABLES THE ADDRESS PINS ON ALL CHIPS ON THE BUS WHEN THE FIRST CHIP OBJECT IS INSTANTIATED!
00025 
00026  USAGE: All Read/Write functions except wordWrite are implemented in two different ways.
00027  Individual pin values are set by referencing "pin #" and On/Off, Input/Output or High/Low where
00028  portA represents pins 0-7 and portB 8-15. So to set the most significant bit of portB, set pin # 15.
00029  To Read/Write the values for the entire chip at once, a word mode is supported buy passing a
00030  single argument to the function as 0x(portB)(portA). I/O mode Output is represented by 0.
00031  The wordWrite function was to be used internally, but was made public for advanced users to have
00032  direct and more efficient control by writing a value to a specific register pair.
00033  */
