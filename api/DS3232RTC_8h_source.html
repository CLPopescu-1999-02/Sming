00001 /*----------------------------------------------------------------------*
00002  * DS3232RTC.h - Arduino library for the Maxim Integrated DS3232        *
00003  * Real-Time Clock. This library is intended for use with the Arduino   *
00004  * Time.h library, http://www.arduino.cc/playground/Code/Time           *
00005  *                                                                      *
00006  * This library is a drop-in replacement for the DS1307RTC.h library    *
00007  * by Michael Margolis that is supplied with the Arduino Time library   *
00008  * above. To change from using a DS1307 RTC to an DS3232 RTC, it is     *
00009  * only necessary to change the #include statement to include this      *
00010  * library instead of DS1307RTC.h.                                      *
00011  *                                                                      *
00012  * This library is *not* a drop-in replacement for the newer version    *
00013  * of the DS1307RTC library at                                          *
00014  * http://www.pjrc.com/teensy/td_libs_DS1307RTC.html                    *
00015  *                                                                      *
00016  * In addition, this library implements functions to support the        *
00017  * additional features of the DS3232.                                   *
00018  *                                                                      *
00019  * This library will also work with the DS3231 RTC, which has the same  *
00020  * features of the DS3232 except: (1) Battery-backed SRAM, (2) Battery- *
00021  * backed 32kHz output (BB32kHz bit in Control/Status register 0x0F),   *
00022  * and (3) Adjustable temperature sensor sample rate (CRATE1:0 bits in  *
00023  * the Control/Status register).                                        *
00024  *                                                                      *
00025  * Whether used with the DS3232 or DS3231, the user is responsible for  *
00026  * ensuring reads and writes do not exceed the device's address space   *
00027  * (0x00-0x12 for DS3231, 0x00-0xFF for DS3232); no bounds checking     *
00028  * is done by this library.                                             *
00029  *                                                                      *
00030  * Jack Christensen 06Mar2013                                           *
00031  * http://github.com/JChristensen/DS3232RTC                             *
00032  *                                                                      *
00033  * CC BY-SA 4.0                                                         *
00034  * "Arduino DS3232RTC Library" by Jack Christensen is licensed under    *
00035  * CC BY-SA 4.0, http://creativecommons.org/licenses/by-sa/4.0/         *
00036  *----------------------------------------------------------------------*/
